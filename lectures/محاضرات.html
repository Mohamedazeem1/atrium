Augmented reality is a type of virtual reality that aims to replicate the real environment in the computer and enhance it with virtual data that was not part of it. In other words, the augmented reality system generates a composite display for the user that blends the real scene viewed by the user and the computer-generated virtual scene that enhances the real scene with additional information.
